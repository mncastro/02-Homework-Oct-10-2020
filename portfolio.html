Coding and Digital Humanities Projects

Omeka Online Exhibit

I coordinated and assembled this bilingual online exhibit for the Benson Latin American Collection in The University of Texas at Austin using the humanities webpage tool OMEKA. Undergrad students curated the materials and label descriptions. All of them form part of the Benson Collection’s treasure trove of rare documents, a collection unlike any other in the world.

DentAPPointment Management System

Second group project for Full-Stack Coding Bootcamp (Tecnológico de Monterrey). This application allows you to register/log in a new user and manage dental appointments for patients. Burger App

A basic application that allows you to create a table with burger names, which you can then eat or erase. This is an advanced assignment for the Full-Stack Coding Bootcamp at Tecnológico de Monterrey. It is meant to showcase a complete Full-Stack application (HTML, Node, NPM Dependencies, Express library, Handlebars HTTP requests, and MYSQL database queries). Application is deployed to an external Heroku server.
